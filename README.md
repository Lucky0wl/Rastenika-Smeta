# Rastenika Smeta

Приложение для автоматизации создания коммерческих предложений (смет) на основе прайс-листов в формате Excel.

## Основные возможности
- Загрузка прайс-листа из XLSX.
- Выбор растений и материалов.
- Автоматический расчет стоимости посадки и налогов.
- Генерация детализированного КП в форматах **XLSX** и **PDF** с сохранением фирменного стиля.

## ⚠️ Важное ограничение
Генерация PDF использует системный API Excel (**win32com**), поэтому приложение должно запускаться на **Windows** с установленным **Microsoft Excel**.

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/Rastenika-Smeta.git
   cd Rastenika-Smeta
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python app.py
   ```

4. Откройте в браузере: `http://127.0.0.1:5000`

## Структура проекта
- `app.py`: Основная логика Flask.
- `pdf_generator.py`: Модуль генерации PDF через Excel COM.
- `templates/`: HTML-шаблоны.
- `static/`: CSS и JS файлы.
- `Шаблон.xlsx`: Фирменный шаблон для выгрузки.
